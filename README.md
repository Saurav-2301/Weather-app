#Weather App
A simple and responsive Weather Application built using HTML, CSS, and JavaScript, following the logic and approach taught in Love Babbar’s Web Development sessions.
The app fetches real-time weather details using a public weather API and displays them in a clean UI.

--Features :-)

Search weather by city name

Shows:

Temperature

Weather Condition

Humidity

Wind speed

Weather icon

Auto-detect weather using user’s location (Geolocation API)

Clean UI + Fully responsive

API error handling (invalid city, empty search, etc.)

---Tech Stack :)

HTML5 – structure

CSS3 – styling

JavaScript (Vanilla JS) – API calls + logic

OpenWeatherMap API – weather data.
